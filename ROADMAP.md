# Roadmap

## v0.1.0 — Core Bookmarking ✅
- Browser extension (Chrome/Firefox/Edge) with popup save UI
- Backend API: CRUD bookmarks, tags, collections
- CLI tool (`bm`) for managing bookmarks
- SQLite database
- Auto-detection platform (GitHub, HuggingFace, arXiv, Reddit, HN, Twitter)

## v0.2.0 — AI Enhancement ✅
- Multi-provider AI support (OpenRouter, NVIDIA NIM, custom proxy)
- Smart summarization
- Auto-tagging
- Semantic search (embeddings + cosine similarity)
- One-click "Smart Save" in extension
- Config management

## v0.3.0 — Projects & Scheduling (Up Next)
- Project management (group bookmarks into projects)
- Scheduling system with calendar view
- Dashboard web UI
- AI-powered priority suggestions
- Progress tracking

## v1.0.0 — Stable Release
- Chrome Web Store & Firefox Add-ons listing
- Cross-device sync
- Import/export browser bookmarks
- Collaboration features
- Performance optimizations
- i18n support
