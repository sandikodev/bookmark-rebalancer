Bookmark Rebalancer

AI-powered bookmark manager for developers. Save, organize, and rediscover your bookmarks with intelligence.

View on GitHub Download Extension Documentation

Features

🌐

Cross-browser Extension

Works on Chrome, Firefox, and Edge. Built with WXT for seamless multi-browser support.

🤖

AI-Powered

Smart summarization, auto-tagging, and semantic search. Supports OpenRouter, NVIDIA NIM, or your own AI proxy.

🔌

Multi-Provider AI

Bring your own AI. Plug in any OpenAI-compatible provider — OpenRouter, NVIDIA, Baseten, or custom.

📂

Smart Organization

Collections, tags, projects, and auto-detection for GitHub repos, HuggingFace models, arXiv papers, and more.

🔍

Semantic Search

Find bookmarks by meaning, not just keywords. AI embeddings power intelligent discovery.

⚡

CLI Tool

Full control from terminal. Manage bookmarks, projects, and schedule with the bm command.

📅

Projects & Schedule

Group bookmarks into projects, track progress, and schedule work sessions. Perfect for OSS contribution planning.

📦

Self-Hosted

Your data, your control. SQLite database, Docker support, no cloud dependency.

Quick Start

🐳 Docker
🟡 Bun
🤖 AI Config
💻 Usage
# Start the backend with Docker docker compose up -d Server running at http://localhost:3000 Data persisted in Docker volume

# Stop when done docker compose down
# Install dependencies bun install # Build shared package bun run packages/shared build # Start backend in dev mode bun run backend:dev Server running at http://localhost:3000

# Or build for production bun run backend:build && bun run backend:start
# Configure OpenRouter (recommended) export BM_API=http://localhost:3000 bun run cli:dev config set-ai \ --key sk-or-v1-xxxxxxxx \ --provider openrouter \ --model openai/gpt-4o-mini
# Or NVIDIA NIM bun run cli:dev config set-ai \ --key nvapi-xxxx \ --provider nvidia \ --model meta/llama-3.1-8b-instruct
# Or custom proxy (OpenAI-compatible) bun run cli:dev config set-ai \ --key sk-xxx --provider custom \ --base-url https://your-proxy.com/v1
# Save a bookmark bun run cli:dev add https://github.com/honojs/hono --smart # List bookmarks bun run cli:dev list --platform github # Semantic search bun run cli:dev search "fast web framework" --semantic # Create a project bun run cli:dev project create "Build API" --priority 5 # Schedule work bun run cli:dev schedule add <project-id> --date 2026-06-01 --duration 90 # Run extension dev mode bun run extension:dev

CLI Reference

📖 Bookmark Management

bm add <url>Save a bookmark
bm add <url> --smartAI auto-summarize & tag
bm list [--platform] [--tag]List with filters
bm search <query>Keyword search
bm search <q> --semanticAI semantic search
bm open <id>Open in browser
bm rm <id>Delete bookmark

📁 Projects

bm project create <name>Create a project
bm project listList all projects
bm project show <id>Project detail
bm project add-bookmark <pid> <bid>Link bookmark to project
bm project rm <id>Delete project

📅 Schedule

bm schedule add <pid> --date <d>Schedule work session
bm schedule list [--project] [--date]List schedule
bm schedule done <id>Mark task complete
bm schedule rm <id>Delete schedule entry

🤖 AI & Config

bm summarize <url|id>AI summarization
bm smart-tag <url>Suggest tags via AI
bm config showShow AI configuration
bm config set-ai --key <k>Configure AI provider
