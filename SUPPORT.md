# Support

## Getting Help

If you need help with Bookmark Rebalancer, here are the available channels:

### Documentation
- [README.md](README.md) — Quick start and CLI reference
- [CONTRIBUTING.md](CONTRIBUTING.md) — Development setup guide
- [ROADMAP.md](ROADMAP.md) — Project roadmap and milestones

### Issues
- [Bug Reports](https://github.com/sandikodev/bookmark-rebalancer/issues/new?labels=bug&template=bug_report.md)
- [Feature Requests](https://github.com/sandikodev/bookmark-rebalancer/issues/new?labels=enhancement&template=feature_request.md)
- [Search Existing Issues](https://github.com/sandikodev/bookmark-rebalancer/issues)

### Community
- GitHub Issues is the primary discussion platform
- For questions, use [GitHub Discussions](https://github.com/sandikodev/bookmark-rebalancer/discussions) (if enabled)

### Before Opening an Issue
1. Check the [existing issues](https://github.com/sandikodev/bookmark-rebalancer/issues) for similar problems
2. Make sure you're running the latest version
3. Include relevant logs and steps to reproduce
